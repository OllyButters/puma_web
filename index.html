Publications Metadata Augmentation (PUMA) pipeline web output

The PUMA pipeline https://github.com/OllyButters/puma/ takes a list of publications (usually academic journal articles) and adds metadata to them so some bibliometric analysis can be done. From this we can automatically build some web pages which displays this information and makes the list of publications easy to explore.
